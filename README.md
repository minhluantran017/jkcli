# jkcli

`jkcli` - stands for 'Jenkins commandline interface' - is a small tool written
with the purpose of integrating Jenkins task calls from GUI to CLI.

## Preperquiste

* Python >= 3.6

## Getting Started

### Installation

```
pip install jkcli
```

### Show help

```
jkcli --help
```
